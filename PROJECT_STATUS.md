# DFN GAN Project Status

Last updated: 2026-06-01

## Current Scope

This project implements a WGAN-GP baseline for generating 2D binary DFN images. The current "reference" dataset is synthetic DFN generated by rule-based line-segment sampling, not field-measured fracture data.

The project is ready for a project presentation baseline:

1. Generate synthetic reference DFN PNGs and metadata.
2. Train WGAN-GP on 128 x 128 binary DFN images.
3. Save generated probability and binary sample grids.
4. Evaluate generated DFNs against the reference synthetic dataset with image-based DFN metrics.

## Current Training Result

The WGAN-GP run completed 200 epochs and produced checkpoints up to `wgan_gp_epoch_0200.pt` plus `wgan_gp_latest.pt`.

Generated samples evolve from noise-like textures in early steps to clear fracture-like line networks after several thousand steps. This indicates that the GAN baseline has learned basic DFN morphology, although generated images still contain artifacts such as extra fragments, curved branches, and local density imbalance.

## Evaluation Result Available

The current evaluation result is:

```text
outputs/evaluation/step_0005000/
```

It compares:

```text
reference: 512 synthetic DFN images from data/synthetic_dfn_128/images
generated: 64 generated images split from outputs/samples/step_0005000_binary.png
```

Key interpretation:

- Pixel ratio is close: generated samples are only slightly sparser than reference.
- Skeleton length is close: generated total line length is in the right range.
- Orientation histogram is close: generated line directions roughly match the uniform-orientation reference.
- Connected components are too many: generated samples are more fragmented than reference.
- Mean component area is much smaller: generated networks break into smaller pieces.

This is a reasonable baseline result for the GAN part: visually and statistically it has learned line-like DFN structure, but connectivity and fragmentation still need improvement.
